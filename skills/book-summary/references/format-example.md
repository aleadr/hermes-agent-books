# Format Example: summary - REALITY+.MD

Contoh struktur file output untuk buku 24+ chapter:

```
# summary - REALITY+

## Preface: Adventures in Technophilosophy

**Judul asli:** Adventures in Technophilosophy
**Judul Bahasa Indonesia:** Petualangan dalam Teknofilsafat

[1 paragraf ringkasan dalam Bahasa Indonesia]

## Chapter 1: Is This the Real Life?

**Judul asli:** Is This the Real Life?
**Judul Bahasa Indonesia:** Apakah Ini Kehidupan Nyata?

[1 paragraf ringkasan dalam Bahasa Indonesia]

...

## Appendix: Notes / Endnotes

**Judul asli:** Notes / Endnotes
**Judul Bahasa Indonesia:** Catatan Akhir

[1 paragraf ringkasan dalam Bahasa Indonesia]
```

Aturan:
- Preface = Introduction/Pengantar (bukan Acknowledgements)
- Appendix = Notes/Catatan Akhir (bukan Index atau Glossary)
- Judul Bahasa Indonesia: terjemahan bebas, esensi sampai
- Summary: 1 paragraf, variasi panjang sesuai isi chapter
- Semua konten WAJIB berdasarkan hasil query RAG — tidak boleh halusinasi
